## How to Use

### For Customers:
1. Visit `/services` page
2. Browse all available services with details
3. Click **"View Details"** to see full service information
4. Click **"Book Service"** to request the service
5. Click **"Send Inquiry"** to contact with questions

### To Seed Services Data:
```bash
npm run seed:services
```

### To Clear Services Data:
```bash
npm run seed:services -- -d
```

| DELETE | `/api/services/:id` | Delete service (Admin) |
